Notatki: Moduł III

Atrybuty w HTML

  • br - nowa linia
  • strong - wyróżnia tekst + podkreśla o istotności (ma znaczenie semantyczne)
  • em - podkreśla inne znaczenie ('i' nie ma znaczenia semantycznego)
  • u - oznaczanie błędów

Linkowanie

  • href='#id' = przewijanie strony do elementu o danym id
  • href + target='_blank' - nowe okno
  • znacznik nav do nawigacji pomiędzy stronami

CSS

!important - kasuje inne reguły css z innych klas
