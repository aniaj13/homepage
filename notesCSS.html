Notatki CSS

Moduł IV

Overflow- decyduje co zrobić z treścią, która wystaje poza boxa - hidden, auto, clip (bardziej zaawansowane), scroll, visible

Jednostki - vw (viewport width) - % widocznego ekranu

box-sizing: border box

Position:

  • static - default przeglądarki
  • relative - tworzy przestrzeń, w której można użyć absolute
  • absolute - wartości top, right, bottom, left - oddalenie od
  • fixed - przykleja element do strony
  • sticky - przyklejanie elementu w obszarze rodzica np. przyklejanie nagłówków - gdy jest overflow to nie zadziała + musi być określona wartość top

z-index - warstwowanie elementów na stronie

float - left, right - opływanie treścią - dobre dla zdjęć w artykułach

clear - czyści floaty

Normalizacja CSS - normalize.css +
html {
box-sizing: border-box;
}
*, *:before, *:after {
box-sizing: inherit;
}

BEM- block element modifier - sposób nazywania klas css - odwzorowanie struktury html w css - nadrzędny element - element - modyfikator
block__element--modifier
klasa + klasa z modyfikatorem

Moduł V

Jednostki:

  • 1 em - font-size dla elementu ustawiony bądź dziedziczony
  • 1 rem - font-size dla root elementu (1 rem = 16px) (definiowany przez html)

Flexbox:

  • flex-wrap = zawijanie wiersza/kolumny - wrap, nowrap, wrap-reverse
  • flex-flow = flex-direction + flex-wrap
  • align-content = używa się przy flex wrap, gdy mamy więcej kolumn/wierszy, ustawianie wierszy i kolumn wg. drugiej osi
  • align-items = ustawianie elementów wzdłuż drugiej osi
  • align-self - do wyłamania elementu ze schematu flex z rodzica - przyjmuje indywidualne wartości
  • order - kolejność elementów
  • flex-grow i flex-shrink
  • flex-basis
  • row-gap & column-gap lub gap

Tła:

  • background-image: url('')
  • background-position: x y
  • background-repeat: repeat; round - tło nie jest ucinane, jest skalowane; space - działa jak flexbox, bez skalowania jak w round ; repeat-x; repeat-y
  • background-attachment: scroll ; fixed - tło nieruchome ; local - scrolluje się wewnątrz elementu (z overflowem) ;
  • background-origin: padding-box, border-box, content-box
  • background-clip: ucinanie tła - padding-box, border-box, content-box
  • background-size: auto ; cover - wypełnia całą przestrzeń; contain - rozszerza się by cały się pokazać ;
