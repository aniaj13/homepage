Notatki CSS

Moduł IV

Overflow- decyduje co zrobić z treścią, która wystaje poza boxa - hidden, auto, clip (bardziej zaawansowane), scroll, visible

Jednostki - vw (viewport width) - % widocznego ekranu

box-sizing: border box

Position:

  • static - default przeglądarki
  • relative - tworzy przestrzeń, w której można użyć absolute
  • absolute - wartości top, right, bottom, left - oddalenie od
  • fixed - przykleja element do strony
  • sticky - przyklejanie elementu w obszarze rodzica np. przyklejanie nagłówków - gdy jest overflow to nie zadziała + musi być określona wartość top

z-index - warstwowanie elementów na stronie

float - left, right - opływanie treścią - dobre dla zdjęć w artykułach

clear - czyści floaty

Normalizacja CSS - normalize.css +
html {
box-sizing: border-box;
}
*, *:before, *:after {
box-sizing: inherit;
}

BEM- block element modifier - sposób nazywania klas css - odwzorowanie struktury html w css - nadrzędny element - element - modyfikator
block__element--modifier
klasa + klasa z modyfikatorem
